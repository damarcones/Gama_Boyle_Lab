Introdução

Este documento tem como finalidade explicar os princípios e diretrizes do projeto Gama Boyle Lab.


Princípios e Diretrizes

Correspondência com as expectativas do usuários

Nosso projeto busca fazer com que um experimento no laboratório de química se torne remoto, e fiel. Para que cumpra aquilo que os alunos esperam ao usar nosso sistema.

Simplicidade nas estruturas das tarefas

Nosso projeto busca simplificar o manuseio de componentes do laboratório.

Equilíbrio entre controle e liberdade do usuário

O Equilíbrio de nosso projeto será feito de modo que os professores e técnicos terão controle sobre a visualização e o uso dos alunos durante o experimento para que assim eles possam prevenir possíveis acidentes.

Consistência e padronização

Nosso projeto usará de uma identidade visual que contará com semelhanças entre funcionalidades, assim então o usuário será sempre levado ao que deve ser, sem se preocupar porque estas são iguais.

Visibilidade e reconhecimento

O nosso sistema será totalmente didático, facilitando assim a execução dos experimentos pelo aluno e seu aprendizado.

Conteúdo relevante e expressão adequada

O sistema web não contará com mudanças da transmissão que o aluno vê do experimento, logo procuramos obter a maior fidelidade possível. Além disso, o projeto não contará com auxilio para deficientes visuais, já que em alguns experimentos é necessário observar a coloração correta da solução.

Projeto para erros

Bem nosso projeto deixará o aluno livre para realizar o experimento sozinho, e para evitar possíveis acidentes os técnicos e professores terão a permissão para pausar ou interromper os experimentos a qualquer momento.
